HTTP Operated Browser

HOB — External AI / script controllable browser

What is HOB?

HOB is a lightweight custom browser built on Neutralinojs. It provides an HTTP API (POST /exec) that allows external AI agents or scripts to execute arbitrary JavaScript inside the browser iframe, and captures all console output to a local log file.

Quick Start

Enter a URL in the address bar above and press Enter to navigate.

From an external terminal, control the browser via HTTP API (port 8080 by default, configurable via HOB_PORT env var):

# Execute JavaScript
curl -X POST http://localhost:8080/exec \
  -H "Content-Type: application/json" \
  -d '{"code": "document.title"}'

# Navigate to a URL (updates the iframe display)
curl -X POST http://localhost:8080/navigate \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

# Get current URL and console logs
curl -s http://localhost:8080/status
curl -s http://localhost:8080/log | tail -10

# Get current page DOM HTML
curl -s http://localhost:8080/dom | head -20
